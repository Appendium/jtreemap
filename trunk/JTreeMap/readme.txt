An example is included in the jar file.

To execute the example :
java -jar jtreemap.jar

To see the applet example :
open the AppletSample.html in your browser

XML FILE
--------
If you want to build your own xml file, see the TreeMap.dtd and the TreeMap.xml
You can create a new xml file which repect the TreeMap.dtd
Then choose File/Open in the JTreeMapExample to load the file.

TM3 FILE
--------
If you want to build your own tm3 file, 
see http://www.cs.umd.edu/hcil/treemap/doc4.1/create_TM3_file.html
The hcil Treemap site explain how to build tm3 file.
Then choose File/Open int the JTreeMapExample to load the file.
